Наиболее популярные языки разметки
HTML Больше всего распространён язык HTML (аббревиатура от английского Hypertext Markup Language − «язык разметки гипертекста»). Он является стандартным для интернет-документов, с его помощью создаются все веб-страницы.
XML XML (в переводе с английского eXtensible Markup Language − расширяемый язык разметки). Название связано с тем, что он не имеет зафиксированного формата.
Wiki Wiki-разметка используется для тех веб-страниц, которые могут правиться (дополняться, редактироваться) пользователями.
